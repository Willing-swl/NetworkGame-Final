# 《充气悖论》(Inflatable Paradox) 完整版游戏设计文档(GDD) - V2 (梦核重构版)

本文档为全量闭环、可直接落地开发的工业级GDD。核心已根据最新的「旧核/商场阈限空间融合」美学设定进行了深度重构与对齐，完全摒弃最初的“马卡龙/泳池核”设定，确保视觉、听觉与硬核博弈玩法的完美自洽。完全对齐Game Jam开发周期与Unity作品集技术展示目标。

## 1. 核心概述 (Overview)

### 1.1 基础信息
| 项目 | 详情 |
|---|---|
| 游戏名称 | 充气悖论 (Inflatable Paradox) |
| 游戏类型 | 本地双人同屏 3D 动作竞技派提示游戏 (Party Brawler) |
| 目标平台 | Windows PC |
| 开发引擎 | Unity 2022.3+ URP管线 |
| 单局时长 | 60秒极速一命制对局 |
| **核心视觉风格** | **梦核 (Dreamcore) / 阈限空间 (Liminal Space)**。<br>视觉核心为「高饱和 PVC 塑料原色（红/蓝/绿/黄/橙），但被黄昏与暗光两层滤镜褪色压暗」。氛围是放学后黄昏与闭店后无人商场的怅然与惊悚反差。 |
| 开发核心目标 | 作为Unity客户端开发作品集，核心展示底层架构（FSM+Event Bus）、渲染性能（MaterialPropertyBlock/顶点形变）、以及深度氛围营造能力，不追求冗余技能数量。 |

### 1.2 核心设计理念
锚定「充气悖论」核心主题：占领地块越多，玩家角色体积越大，受击判定框越广。 玩家必须在“扩张领地拿资源”与“消耗地块缩小体积求生存”之间高频决策。
利用**反差恐怖**（柔软的童真环境 vs 残酷的生存厮杀）营造心理压迫感，打造「低门槛上手，高上限博弈」的体验。

### 1.3 赛制规则
* **核心规则**： 一命淘汰制（1-Stock Elimination），玩家被击飞出斗技台边界即判定淘汰，对方获胜。
* **时长规则**： 单局固定60秒，倒计时结束双方均未淘汰时，按己方领地地块占比判定胜负；占比完全一致时，直接判定为平局。
* **对局规则**： 支持本地双人无限续局，无局外成长系统，单局内所有资源、状态对局结束后清零。

## 2. 核心玩法与循环设计

### 2.1 三层核心循环设计
| 循环层级 | 核心内容 | 设计目标 |
|---|---|---|
| 乐趣原子单位（玩家每秒操作） | 移动瞄准→主动按键喷射颜料涂色/反涂地块→即时获得地块控制权与充气资源反馈 | 让玩家每一秒都有明确操作目标，即时反馈拉满，无空窗期 |
| 单局核心循环 | 占领地块（角色体积变大/风险升高）→ 消耗地块释放冲击波（击飞对手/自身瘦身降风险）→ 地形压迫+累计受伤值放大击飞效果→ 击杀淘汰 | 形成闭环博弈，每一步操作都有明确收益与对应风险，完全贴合「充气悖论」物理主题 |
| 元游戏（玩家每次会话目标） | 本地双人连胜挑战、刷新最短击杀时长本地记录、尝试不同博弈套路 | 解决派对游戏复玩率问题，无需复杂局外系统即可提供长线目标 |

### 2.2 MDA框架机制-动态-美学对应表
| 核心机制 | 玩家行为动态 | 目标美学体验 |
|---|---|---|
| 充气涂色+物理体积膨胀 | 玩家在「扩张领地拿资源」和「消耗地块瘦身降风险」之间反复博弈 | 悖论式决策乐趣、持续的风险对冲紧张感。**角色巨大化时，移动音效愈发沉闷刺耳，强化生理压迫感**。 |
| 分段式领地冲击波（消耗地块） | 玩家需要权衡「用多少资源换攻击强度」，满蓄力高风险操作带来高回报 | 即兴发挥的爽感、赌徒式博弈乐趣。释放瞬间带来“放气”的解脱感。 |
| 分级闪避+完美闪避攻防逆转 | 新手靠普通闪避保命，硬核玩家靠完美闪避实现绝地翻盘 | 操作成就感、逆风翻盘的情绪峰值 |
| 分级动态地形塌陷 | 玩家操作空间持续压缩，从试探到决战的节奏清晰 | 持续升级的压迫感、极速决战的肾上腺素飙升 |
| 充气软体质感顶点Shader | 玩家移动、涂色时地块有实时形变反馈，带有PVC材质反光 | 梦核风格的诡异柔软感与廉价塑料的真实感 |

## 3. 关卡、光照与全景设计 (双梦核融合方案)

采用**双场景融合（商场阈限空间 + 市井旧核）**的方案，将竞技所需的聚焦区与氛围所需的远景完美结合。

### 3.1 物理交互区（核心斗技台）
* **尺寸与形状**： 紧凑的 11x11 切角网格阵列，开局提供合理的战略缓冲空间。
* **核心结构**：
  * **中央高台**：提供高度物理抛物线优势。
  * **四角立柱（不可移动刚体）**：绝对静止。玩家利用其物理碰撞体来阻断敌方冲击波视线。
* **光照设计（商场暗光环境为主）**：
  * 取消常规全局漫反射光。
  * **点光源阵列**：在斗技台地平线边缘布置微弱的暖白/暖橙点光源，自下而上照射地块和立柱，制造非正常的局部高亮和深邃阴影，确保暗环境下的高对比度竞技可读性。

### 3.2 视觉远景区（记忆深渊）
* **设计内核**： 斗技台如同漂浮在一个巨大、空旷的废弃记忆空间内。
* **视觉元素**：
  * **超尺度静物**： 深渊四周潜伏着巨大的充气滑梯、独角兽轮廓，只有面向主角的一侧被微弱边缘光照亮，背面全黑。
  * **市井暮光远景**： 极远处的地平线上，隐约可见一排老旧居民楼的剪影（黄昏旧核）。
  * **暖灰褐迷雾**： 雾色（`#484038`）类似黄昏烟尘与夜市油烟，深度分层（0-15米无雾/清晰，15-30米线性增浓，30米切黑）。

### 3.3 60秒动态地形塌陷节奏
| 时间段 | 阶段名称 | 地形变化规则 | 核心玩法引导 |
|---|---|---|---|
| 0-20秒 | 充气试探期 | 完整11x11地图，无塌陷 | 玩家自由抢地，积累充气资源，熟悉对手操作节奏。 |
| 20-35秒 | 空间施压期 | 最外围一圈网格进入「漏气预警」（闪烁+漏气音效），35秒坠入深渊 | 玩家权衡退守，提前规划安全区。 |
| 35-50秒 | 充气决战期 | 第二圈网格进入漏气预警，50秒坠入深渊，场地大幅收缩 | 正面博弈，利用冲击波地形杀。 |
| 50-60秒 | 极速猝死期 | 每隔1.5秒随机掉落1块非占领地块，同时启动最高级“滤镜污染” | 杜绝平局拉扯，情绪推至最高潮。 |

## 4. 核心玩法机制详解

### 4.1 状态与动态体积归属
* **地块颜色（核心变更）**： 放弃马卡龙，采用高对比PVC原色。Neutral（老化米黄白 `#D8D0C0`）、**P1（充气城堡蓝 `#2E8BC0`）**、**P2（充气城堡红 `#E85040`）**。
* **反占领成本**： 中立地块单次命中0.15秒涂满，敌方地块0.25秒涂满。
* **体积膨胀悖论**： 场上己方领地占比越大，角色体积和碰撞体等比放大（最高1.5倍）。体积越大，受迫音效改变（越沉闷吃力），中弹受击率大幅飙升。

### 4.2 主动喷射涂色机制
* **无限喷射（无弹药槽）**： 颜料无任何消耗限制。涂色本身是获取资源的唯一手段。注意力集中在抢地与走位。
* **精度与衰减**： 最大有效射程3格。奔跑状态距离-1且散射增加50%；站立/行走为完全精准。

### 4.3 攻防博弈系统
#### 4.3.1 分段式领地冲击波
* 按住重击键蓄力（吸收脚下及相邻最多8格），释放引爆。蓄力期锁定移动、喷射与闪避。
* **Lv.1 (1-3格)**：基础击退；**Lv.2 (4-6格)**：中等冲击+硬直；**Lv.3 (7-8格)**：极大半径与极强击飞。
* 释放后，角色体积与受击框瞬间缩小。

#### 4.3.2 分级闪避系统
* **普通闪避**： 0.2秒无敌帧，固定位移，冷却0.8秒。
* **完美闪避**： 受击前0.15秒内按下。无视冲击并瞬间触发子弹时间，周围九宫格同化为己方颜色。

#### 4.3.3 累计受伤值运动学击飞
* **受伤值累加**： 被冲击波命中累加（Lv.1+15%，Lv.3+120%）。
* **击飞公式**： 击飞距离 = 基础推力 × (1 + 受伤值/100)^1.5。所有击飞采用运动学精密计算（精准抛物线物理）完全替代不可控的布娃娃（Ragdoll）引擎。

## 5. 玩家控制输入系统
全局同屏显示（无分屏）。基于Unity New Input System。强制10帧输入缓存。

### 5.1 双手柄模式 (唯一目标设计基准)
| 操作功能 | 通用手柄键位 | 输入逻辑与判定规则 |
|---|---|---|
| 角色移动 | 左摇杆（L-Stick） | 全方向精准移动，死区20% |
| 视角/瞄准 | 右摇杆（R-Stick） | 控制角色朝向与自由无锁定喷射方向 |
| 颜料喷射 | 右扳机键（RT/R2） | 按住持续喷射，随时可松开 |
| 重击蓄力 | 左扳机键（LT/L2） | 按住吸取地块蓄力，松开轰击释放 |
| 闪避/滑铲 | 左/右肩键（LB/RB） | 食指控制，确保拇指在此过程不离开操作摇杆 |
| 暂停游戏 | 菜单键 | 冻结全局时空 |

## 6. 美术材质与后处理规范 (Dreamcore 重构版)

### 6.1 材质与色彩属性
* **充气塑料材质**： 所有可交互物体（地块、立柱、角色）均采用 Metal=0, Smoothness=**0.65~0.75** (强力反光)。表面贴图增加微弱法线扰动（模拟 PVC 焊接缝和磨损老化）。
* **P1与P2角色**：采用比领地色更深的“印刷色”（像充气堡上粗糙印刷的卡通小人，带有一定的失真感）。

### 6.2 Shader & 后处理管线 (事件驱动动态滤镜)
| 模块 | 核心逻辑 | 效果目标 |
|---|---|---|
| **顶点形变** | MaterialPropertyBlock传参距与下陷权重计算。 | 还原充气床物理下陷软弹效果，形变回弹跟随反光视角变化。 |
| **角色受挫膨胀** | 表面沿法线向外偏移伸展，绷紧材质反光率。 | 角色巨大膨胀带来视觉上的压迫和即破的气球紧绷感。 |
| **天光抑制方案** | 采用低色温斜角主光 + 冷蓝补光的混合压制。 | 重塑黄昏场景过渡到闭店黑夜的疏离感。 |
| **动态记忆滤镜** | **极其关键的交互体现**：<br>1. 常规全量对局（0-50s）：保持画面干净可读，仅开启微弱Film Grain与极微度Color Grading。<br>2. 决战收缩阶段（50-60s）：屏幕边缘微弱色散（Chromatic Aberration）缓慢上升。<br>3. 峰值冲击（大招/淘汰）：全屏激烈的色差抽帧抖动（持续0.5秒）随后恢复。 | 不损失竞技类游戏高操作视野的情况下瞬间点燃且撕裂梦核深渊的恐惧错乱感。 |

## 7. 音效与视听错位设计规范 (Audio Detachment)
**核心原则**：摒弃传统格斗“正向反馈（例如热血的爆炸砰声）”，统统替换为了极其干瘪、沉闷却压抑的**“日常物理噪音”**来进行精神暗示摧毁。

| 互动状态 | 音效设计基准 | 情感传递 |
|---|---|---|
| 颜料喷射 | 沉闷刺耳的劣质塑胶相互摩擦声 | 动作压抑的徒劳无功感 |
| 冲击波蓄力 | 远处鼓风机高频压抑运转的声音，带轻微耳鸣 | 即将爆炸前的临界焦虑 |
| 冲击波释放 | 一声突兀、巨大且非常干瘪的“噗嘶——”猛烈放气声 | 瞬间解脱的空虚与虚无 |
| 完美闪避 | 极不和谐的强行突刺音（比如一段走调的夜市儿歌电音或突如其来的泳池滴水混响） | 打破安全区，造成阈限认知失调恐慌 |
| 受飞淘汰时 | **环境底噪全部骤停半秒（如断电失聪）**，紧接着是角色干瘪如橡胶小黄鸭般的虚弱哀嚎 | 堕入无望深渊的绝对死寂感 |
| 全局环境底噪 | 后台永远存在的低频风噪 + 极遥远模糊不清的空旷建筑录音（含糊的超市广播或人群白噪音） | 没有活人却带有生活痕迹的孤独感 |
| **角色巨大化** | 角色移动速度本身*不减缓*，但移动步伐的物理摩擦音轨变粗重、刺耳 | 视觉与听觉同步营造生理挤压感 |

## 8. 技术程序架构约束
* **MVC剥离**：严禁 MVVM 等双向数据绑定强耦合机制。强制采用底层基于事件发送驱动引擎（Event Bus）+ 分层级的系统管理器（Manager）进行全量逻辑解耦搭建。
* **物理剥离**：所有参战角色主体只有唯一的物理特征—只做碰撞检测（胶囊体 Collider拦截）。人物位移、硬直动画切换、抛物线落地拦截全都基于角色状态机直接计算预判和位移插值重载赋值，**杜绝使用自带 Ragdoll 带来的穿墙反物理BUG。**
